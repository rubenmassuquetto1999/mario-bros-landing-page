# 🍄 Mario Bros - Landing Page

Uma landing page responsiva, moderna e interativa desenvolvida para os encanadores mais famosos do Reino dos Cogumelos: Mario e Luigi! 

Este projeto foi construído utilizando o conceito de **Single Page Application (SPA)**, permitindo uma navegação fluida entre seções sem recarregar a página, o que mantém o vídeo de fundo rodando continuamente de forma imersiva.

🔗 **[Clique aqui para ver o projeto online](https://rubenmassuquetto1999.github.io/mario-bros-landing-page/)** *(Assim que o GitHub Pages finalizar o deploy)*

---

## 🚀 Funcionalidades

* **Navegação SPA:** Transição suave entre a "Home" e "Nossos Serviços" usando JavaScript puro, sem dar reload na página.
* **Vídeo de Fundo (Background Video):** Vídeo em loop com uma máscara de gradiente escuro (`linear-gradient`) para garantir a legibilidade do texto.
* **Formulário Dinâmico:** Formulário de contato flutuante com efeito deslizante ao clicar nos botões de "Fale Conosco" ou "Pedir orçamento".
* **Envio de Formulário Funcional:** Integração com a API do **Formcarry** para receber os orçamentos sem necessidade de um backend.
* **Integração com WhatsApp:** Botão flutuante no canto inferior direito que redireciona o usuário diretamente para o WhatsApp com uma mensagem pré-definida.
* **100% Responsivo:** Layout adaptável para Desktop, Tablets e Celulares (Android e iOS), utilizando *Media Queries* e *Flexbox*.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização, variáveis de estado flutuantes (`translate`, `calc()`), *Media Queries* e *Flexbox*.
* **JavaScript (Vanilla):** Manipulação do DOM para alternância de telas (SPA) e controle da interface do formulário (classes CSS).
